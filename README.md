
# 大丰收食材批发配送有限公司 (Great Harvest Fresh)

![大丰收 Logo](https://img.icons8.com/color/96/harvest.png)

## 📖 品牌背景
**大丰收** 是菲律宾顶级食材供应链平台，由 **菲律宾江西商会** 荣誉背书。

项目于 2019 年在商会创始人 **许俊华先生** 带领下创立，目前由商会促进经济会长 **邱总** 亲自担任董事长。我们致力于为菲律宾政府机关部门、军队食堂以及国内在菲华人企业提供高标准的食材批发配送服务，拥有完善的资质与极高的行业信誉。

## ✨ 核心亮点
- **商会信誉保障**：每一份食材均由江西商会品质小组严选监制，邱总亲自把关。
- **政企供应标准**：符合政府与军队集采的严格检疫、发票合规及溯源要求。
- **实时询价体系**：针对生鲜市场价格波动，系统集成 Telegram 实时行情对接，确保价格公开透明。
- **全马尼拉鲜配**：覆盖 90+ 种精选肉类、海鲜及季节性蔬菜，凌晨采摘，当日送达。

## 🛠 技术架构
- **核心框架**：React 19
- **样式方案**：Tailwind CSS (响应式布局)
- **构建工具**：Vite 6 (针对 Vercel 优化)
- **部署环境**：Vercel / GitHub Pages
